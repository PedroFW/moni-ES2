# moni-ES2
Sistema de monitorador de gastos para trabalho prático da disciplina de Engenharia de Software II

### Instruções

backend

instale o flask,
```
pip install Flask
```
rode,
```
python app.py
```

frontend

instale o vue,
```
npm install -g @vue/cli
```
prepare,
```
npm install
```
rode,
```
npm run serve
```
